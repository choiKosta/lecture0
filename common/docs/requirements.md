# Common 요구사항

## 책임 범위

Common 영역은 Client와 Server가 공유해야 하는 인터페이스 계약, 공통 요구사항, 오류 모델 문서를 관리한다.

## 공통 관리 대상

- 제어 명령 명세
- 제어 응답 명세
- 상태 코드
- 오류 코드
- RTSP URL 형식
- Frame Rate 지원 범위
- Resolution 지원 목록
- 버전 호환성 정책

## 원칙

- Client 전용 UI 로직을 포함하지 않는다.
- Server 전용 스트리밍 구현을 포함하지 않는다.
- Client 또는 Server의 빌드/IDE 설정을 포함하지 않는다.
- 특정 언어에 종속된 구현 코드를 포함하지 않는다.
- 양쪽 프로젝트가 함께 합의해야 하는 계약만 포함한다.
- 계약 변경 시 Client와 Server 영향도를 함께 기록한다.
