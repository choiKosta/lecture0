# Server Project

카메라 관제 Server 프로젝트 영역입니다. 이 README는 Server 영역에 어떤 문서와 코드가 있는지만 안내합니다. 기능, 개발환경, 빌드, 검증 기준의 구체 내용은 요구사항 문서에서 관리합니다.

## Directory Contents

- `src/`: Server 구현 코드
- `docs/`: Server 요구사항 및 설계 문서
- `build.gradle`: Server Gradle 빌드 설정
- `settings.gradle`: Server Gradle project 설정

## Documents

- [Server 요구사항](docs/requirements.md)

## Ownership

Server 팀이 소유합니다. Client 구현, Client 검증 절차, 공통 계약은 이 영역에서 관리하지 않습니다.
